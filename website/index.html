How It Works

Three steps.
That's it.

PluginSmith follows a simple workflow. Import your source material, let AI help you configure the plugin structure, and generate everything with one click.

1

Import

Drop in Markdown, PDF, Word, Excel files, or paste a URL. PluginSmith parses and analyzes your documentation automatically.

2

Configure

Define commands, skills, agents, and hooks. Or let AI analyze your sources and suggest the entire plugin structure for you.

3

Generate

One click generates all files. Install directly to Claude's local marketplace, export to a folder, or package for distribution.

Setup

Bring your own
Claude API key.

PluginSmith uses the Anthropic API to analyze your documents and generate plugin content. You'll need your own API key. Your key stays on your machine, stored securely in the macOS Keychain.

Get your API key

It only takes a minute. Here's how.

  1. Create an Anthropic account Go to console.anthropic.com and sign up or log in.
  2. Generate an API key Navigate to Settings → API Keys and create a new key.
  3. Copy the key It starts with sk-ant-. Copy it to your clipboard.
  4. Paste into PluginSmith Open PluginSmith → Settings → paste your API key. The app validates it instantly.

Why is it free?

PluginSmith is an open-source tool built by a developer, for developers. It's released under the MIT License because great tools should be accessible to everyone. The only cost is your own Anthropic API usage, which you control entirely. No accounts. No tracking. No subscriptions. Just a tool that works.
